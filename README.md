# EdLab Node.js Skeleton

## Development
### Start development server
```
npm i
npm run dev
```

### Testing
```
npm run test
npm run test:report
npm run test:coverage
```

### Bump version
```
npm run bump
npm run bump:minor
npm run bump:major
```

## Deployment
### Build
```
npm run build
```

### Start server under docker
```
npm run start:docker
```
